Semantic Elements

Tag Name Purpose
<article> Article Stand alone content that makes sense alone. Example: Blog, comments, and magazines.
<section> Section Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<aside> Aside Marks additional information that can enhance another element but isn't required to understand it.
<header> Header Represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also a logo, a search form, an author name and other elements.
<nav> Navigation Represents a section of the page whose purpose is to provide navigation links.
<main> Main Represents the dominant part of the <body> content.
<footer> Footer Typically contains info about the author of the section, copyright data or links to related documents.

CSS Selectors

Selector Purpose Example Explanation Specificity Rank
Type Selector All appeareances of the selected element in index.html will be affected by the type ruleset. h1 {
color: blue; }		 Applies the color blue to all h1 elements in your index.html file. 0 (Lowest)
Universal selector This selectors ruleset applies to all elements on the webpage. * {
background-color: black}		 The symbol "*" applied the shown ruleset to the entire webpage, turning the entire page black. 0
Class selector Applies to all elements that are apart of the selected class. All class selectors in CSS have a preceding "." that signals to the code that a class is present. Elements can contain more than one class and multiple elements can have the same class. .paragraphs {
}		 The ruleset to the left is applied to all elements that contain the class "paragraphs" 1
Attribute selector Targets all elements that contain the selected attribute. Can be as specific or non-specific as you want. [href] {
}		 The ruleset is applied to all elements that contain the "href" attribute. 1
Psuedo selector Applies the ruleset to the selected element when a certain criteria is met. a:visited {
color: purple;
}		 "a" stands for the element <a> which is an anchor tag that allows us to specify a link to text. The :visited you see in the example makes it so that once the link is clicked and opened the ruleset is activated, thus turning the link purple. 1
ID selector Applies the ruleset to the element that contains the selected ID. Can only have a single value and can only be used once per page. #header {
}		 The "#" is used to signify to the file that an ID is being used. The following ruleset is applied to the element that contains the ID: 'header' 2 (Highest)

CSS Properties

Property Used for Example Explanation
Color Changes the color of the selected elements h1 {
color: red;
}		 All h1 elements in index.html were turned red by the property color
Background-color Changes the color of the background of the selected element div {
background-color: blue;
}		 This changes the background-color of all div's in the webpage to blue.
Font-size Changing the size of the font. Its units are pixels, rems and ems.' p {
font-size: 20px;
}		 The font-size of all paragraph (p) elements are changed to 20px (pixels)
Font-family Selecting the style of font you want to use. h2 {
font-family: 'Arial', sans
}		 All h2 elements in the webpage have their font changed to Arial. If Arial doesn't work due to browser compatibility then sans will be used as a backup.
Font-weight Allows you to manipulate how bold or light an element is. Has several values including "lighter", "normal","bold", and "bolder." It is scaled from 0 to 900, where 0 is the lightest and 900 is the boldest. p {
font-weight: bold;
}		 Bolds all the text in paragraph elements on the webpage.
Border Allows you to design the border of the content area. You can custimize the size, fluidity, and color of the border. .redbord {
border :1px solid red;
}		 All elements on the webpage that contain the class "redbord" have a 1 pixel solid red border applied to it.
Padding Allows you to manipulate the space between the content and the border. A very important tool for manipulating the size and placement of elements. selector {
padding: 20px;
}		 All chosen elements will be given a padding on all sides of the element of 20 pixels.
Margin The amount of space between the border and the outside edge of the element. Allows you to manipulate elements in space outside of the selected element. selector {
Margin: 20px 30px;
}		 A vertical margin of 20px was applied to the top and bottom of the selected element. A horizontal margin of 30px was applied to the left and right side of the selected element.
